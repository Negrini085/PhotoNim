The ray tracing code "PhotoNim" facilitates the conversion of images from .pfm to .png format and the rendering of intricate scenes composed of geometric figures. Consequently, it is essential to implement types and functionalities that enable efficient image processing. Within this code, we employ RGB color encoding: the color of a pixel is defined by a triplet of numbers.
